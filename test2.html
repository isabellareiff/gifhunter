Test Page Two

A slightly more complicated web page for testing the CS484 Web Server Programming Project. This page requires two requests from the server. The page includes such interesting HTML elements as:

  • Embedded and inline styles
  • Bold and italic text
  • Unordered list
  • Colored text
  • Colored background
  • A White Walker! (note: White Walker image is in a subdirectory!)
Even with a single thread and unqueued socket connection requests, this page should serve up fine. It wouldn't hurt to either thread the server or, at at the very least, allow a few queued connections on your server to be sure this page comes up properly.

Your server should also serve up a video just fine, like this example of how not to receive your helicopter.

Even JavaScript should work.
